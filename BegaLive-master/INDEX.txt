📞 QUICK TROUBLESHOOTING
========================

Problem: Map not loading
→ Check internet (needs OpenStreetMap tiles)
→ Try opening from web server (not local file)

Problem: Charts not displaying
→ Ensure Chart.js CDN is accessible
→ Check browser console for errors

Problem: Port already in use
→ Try different port: python -m http.server 8001

Problem: "CORS" errors
→ Use web server instead of opening file directly
→ Or use: npx http-server --cors

Problem: Markers not showing
→ Check browser console
→ Verify JavaScript is enabled
→ Try refreshing page
